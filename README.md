# Flight Telemetry Data Pipeline

This project simulates a real-world data engineering scenario inspired by avionics and SCADA systems.

## Objective
Design an end-to-end data pipeline to ingest, validate, transform, and analyze flight telemetry data coming from multiple subsystems.

## Systems Simulated
- Flight system
- Battery system
- Control system
